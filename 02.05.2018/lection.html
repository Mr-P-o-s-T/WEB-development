Java Script - мова програмування яка інтерпретується web-брауіером В основу покладено дії які виконуються над об'єктами html. Крім того вона має власні об'єкти такі як array, data, math, string... Також підтримується ООП. Кожен об'єкт має властивості: певиний набір характеристик, для нього визначені певні дії (методи) завдяки яким можна зміювати його властиості. Всі об'єкти об'єднуються у класи, які створюють певну ієрархічну струектуру за принципом наслідування. За допомогою js можна динамічно змінюваи властивості об'єктів під час певної події.
